# Templete Method Pattern

Frame work과 Library의 차이
프레임웍은 흐름이 모두 정해져있음
라이브러리는 필요 기능을 가져와서 직접 흐름을 구성한다

템플릿 메서드도 프레임웍에서 많이 사용되는 패턴

틀이 갖춰진 메서드.
시나리오에 따라 짜여진 메서드들.
중요한 시나리오 흐름을 가진 메서드므로 final로 선언해 재정의 불가능하게 사용. 하위에서 상속하는 것도 불가능
